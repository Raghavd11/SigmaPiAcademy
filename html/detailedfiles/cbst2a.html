CYBER SECURITY

Semester III - Detailed Syllabus

Unit Topics
Unit 1: Web Security
1.1: Basics of Web
• HTTP
• Static and Dynamic Web Pages
• Layers of the Web Stack: Client-Side, Server-Side, and Database

1.2: Web Browser Security
• Web Browser Components: Rendering Engine, JavaScript Engine, Networking Stack
• Vulnerabilities: XSS, CSRF, Clickjacking
• Security Features: Same Origin Policy (SOP), Content Security Policy (CSP), Sandboxing
• Cookies
• Privacy Settings, Cookie Handling, Pop-up Blockers
• Addons and Plugins

1.3: Secure HTTP
• Risks of HTTP: Data Interception, Eavesdropping, Man-in-the-Middle Attacks
• Role of HTTPS in Securing Communication
Unit 2: Malicious Software
2.1: Malware Types
• Virus, Worms, Trojans
• Spyware, Adware
• Key-loggers, Ransomware

2.2: Malware Propagation
• Email Attachments, Malicious Websites, Removable Media
• File Sharing Networks, Malvertising, Software Vulnerabilities
• Watering Hole Attacks, Botnets

2.3: Protection Against Malware
• Antivirus/Antimalware Tools
• Software Updates, Email/Web Filtering
• Least Privilege, Network Segmentation
• Backup & Recovery, Strong Passwords, MFA
Unit 3: Mobile Device Security
3.1: Types of Mobile Devices
• Mobile Phones, Tablets, Wearables

3.2: Privacy Concerns and Data Collection
• Location Tracking, Geolocation Data
• Device Identifiers (UDIDs)
• Personalized Ads & Monetization Practices
• Data Sharing Risks by Apps and Providers

3.3: Mobile App Security
• Security Concerns with Mobile Apps
• App Permission Management
• Risks from Location-Based Social Apps

3.4: Data Security on Mobile Devices
• Sensitive Data Protection
• Risks of Unencrypted Storage/Communication
• Device Encryption, Secure Messaging, Encrypted Storage

3.5: Network Security Risks
• Risks of Unsecured Wi-Fi, Public Hotspots
• Man-in-the-Middle Attacks, Wi-Fi Spoofing

3.6: Physical Security Threats
• Theft, Unauthorized Access
• Protection Strategies: Passcodes, Biometrics, Remote Wipe, Secure Carrying

3.7: Safe Disposal of Mobile Devices
• Proper Disposal Techniques to Protect Data

Miscellaneous
