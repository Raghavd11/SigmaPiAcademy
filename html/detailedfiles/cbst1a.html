Semester I - Detailed Syllabus

Unit Topics
Unit 1: Computer Systems
1.1: Evolution of Computers
• Generations of Computers
• Brief Introduction to Quantum Computers

1.2: Computer Organization
• Logic Gates and Truth Tables: AND, OR, NOT, XOR
• Functional Components and Interconnections
• Memory Organization (Diagrams Only), Types
• I/O Devices

1.3: Encoding Schemes and Number Systems
• ASCII, EBCDIC
• Binary, Octal, Hexadecimal Number Systems

1.4: Data and Information
• Definitions
• Data vs. Information (with examples)
• Types of Data

1.5: Operating Systems
• Roles and Functions
• Types of OS
• Concepts: Processes, Threads, Memory Management
• File Systems

1.6: Database Management Systems (DBMS)
• Importance of Databases
• Role of DBMS
• Relational Databases
• Introduction to SQL

1.7: Programming a Computer
• Algorithms (Pseudocode), Flowcharts
• Compiler vs Interpreter
• Programming Languages: C, C++, Python, Java, JavaScript
• Python Programming (Simple Introduction)
  – Installation, Basic Syntax
  – Conditionals, Loops, Arrays/Lists/Sets, Functions
Unit 2: Computer Networks
2.1: Types of Networks
• LAN, MAN, WAN, Wireless LAN, Internet

2.2: Components of a Network
• Servers, Workstations, NICs
• Guided Media: UTP, STP, Co-axial, Fibre Optic
• Unguided Media: IR, Radio, Microwave, Satellite
• Devices: Repeaters, Hubs, Bridges, Switches, Routers, Gateways

2.3: Network Topologies
• Mesh, Ring, Bus, Star, Tree, Hybrid

2.4: Channel Concepts
• Channel, Bandwidth (Hz, KHz, MHz)
• Data Transfer Rate (bps, Kbps, Mbps, Gbps, Tbps)

2.5: The Internet
• History and Evolution
• TCP/IP Stack and Protocols
• MAC Address
• IPv4 Classes A, B, C
• Subnet Mask and Default Gateway
• IPv6 (Basic Format)
• ICMP

2.6: Internet Applications
• E-mail, WWW, Domain Name Systems

2.7: Internet of Things (IoT)
• IoT Architecture
• IoT Devices: Sensors, Actuators, Gateways
• IoT Protocols: MQTT, CoAP, Zigbee

2.8: Cloud Computing
• Cloud Service Models: IaaS, PaaS, SaaS
• Deployment Models: Public, Private, Hybrid
• Overview of Cloud Storage
• Major Providers: AWS, Azure, Google Cloud
Unit 3: Introduction to Cybersecurity
3.1: Overview of Cybersecurity
• Relevance in the Modern World

3.2: History of Cybersecurity
• Major Incidents and Their Impact

3.3: CIA Triad
• Confidentiality, Integrity, Availability

3.4: Key Terms and Concepts
• Security, Privacy, Threats, Vulnerabilities
• Exploits, Risks, Attacks, Attack Vectors
• Hackers vs. Crackers

3.5: Cyber Threats and Classification
• Malware
• Social Engineering
• DoS/DDoS
• Insider Threats
• Advanced Persistent Threats (APTs)
• Data Breaches, Information Theft
