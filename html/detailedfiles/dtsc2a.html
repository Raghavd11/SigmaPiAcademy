DATA SCIENCE

Semester III - Detailed Syllabus

Unit Topics
Unit 1: Foundation of Statistics for Machine Learning
1.1: Statistical Foundations
• Distance between distributions: Euclidean norm, Pearson's correlation coefficient
• Basic concepts of chi-square distance (not in detail)
• Bayes' theorem and Bayesian probability

1.2: Vector Algebra
• Real n-dimensional space (ℝⁿ) and vector algebra
• Dot product of two vectors
• Vector projections

1.3: Statistical Measures
• Product moment correlation coefficient (Pearson's coefficient)
• Chi-square usage in measuring distribution distance
• Conditional probability and Bayes’ theorem
• Concept of conditional independence
Unit 2: Introduction to Machine Learning
2.1: Fundamentals of Machine Learning
• What is Machine Learning?
• Traditional programming vs Machine Learning
• Relationship between ML and AI
• Applications of Machine Learning
• Why machines need to learn rather than being designed

2.2: Learning Types and Regression
• Types: Supervised, Unsupervised, Semi-supervised, Reinforcement Learning
• Concepts: training, testing, validation
• Training examples, hypothesis representation, hypothesis space
• Learning Requires Bias, Loss function
• Iterative training in linear regression
• Gradient descent algorithm
• Disadvantages of iterative method
• Learning rate and loss reduction
• Importance of feature scaling (min-max normalization)

2.3: Features in Machine Learning
• What is a feature or attribute?
• Features in different data types: structured, text, image
• Types of features: continuous, categorical
• Representing training examples with multiple features
• Linear regression with multiple attributes
• Feature crossing, polynomial regression
Unit 3: Supervised Learning
3.1: Regression vs Classification
• Difference between regression and classification
• Real-world classification examples
• Linear classification, threshold classifier
• Input space and linear separator
• Drawback of threshold classifier
• Logistic function and hypothesis function
• Logistic regression for binary classification
• Probabilistic interpretation of logistic regression output
• One-vs-all strategy for multi-class classification

3.2: Probabilistic Classifiers
• Bayesian Learning and conditional independence
• Naive Bayes Classifier

3.3: Classifier Performance
• Confusion matrix
• True Positive, True Negative, False Positive, False Negative
• Error, Accuracy, Precision, Recall
• F-measure, Sensitivity, Specificity
• K-fold cross validation
