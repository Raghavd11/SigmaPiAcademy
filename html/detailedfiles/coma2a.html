COMPUTER APPLICATION

Semester III - Detailed Syllabus

Unit Topics
Unit 1: Python Programming
Basics of Python:
•Introduction, features, modes (interactive & script), character set, tokens, variables, comments.
Data Types:
Numbers, Boolean, sequence (string, list, tuple), None, dictionary, mutable & immutable types.
Operators:
•Arithmetic, relational, logical, assignment, augmented, identity, membership.
Expressions, Type Conversion & I/O:
•Operator precedence, expression evaluation, explicit/implicit conversion, console input/output.
Errors:
•Syntax, logical, and run-time errors.
Flow of Control:
•Indentation, sequential, conditional, and loop flow.
Conditional Statements:
•if, if-else, if-elif-else.
Loops:
•for, while, range, break, continue, nested loops.
Strings:
•String operations, traversal, slicing, methods (e.g., len(), lower(), find(), split(), join()).
Lists:
•Indexing, list methods (append, insert, remove, etc.), nested lists.
Python Modules:
•Importing modules, math, random, statistics modules and their functions.
Functions:
•Built-in, user-defined, parameters, return values, scope (global/local), flow of execution.
Unit 2: E-Commerce
Introduction to E-Commerce:
•Meaning, goals, activities, scope, advantages/disadvantages, types (C2C, B2B, etc.), internet/intranet/extranet, B2B apps, banners, web promotion.
Electronic Payment System:
•Types of e-payment, credit cards, fund transfer, paperless billing, e-cash, value exchange systems.
Internet Marketing:
•Pros & cons of online shopping, marketing cycle, personalization, techniques, justify internet business.
