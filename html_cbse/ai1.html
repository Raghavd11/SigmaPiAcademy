Artificial Intelligence – CLASS IX
Unit Topics
Unit 1: AI Reflection, Project Cycle and Ethics
• Introduction to Artificial Intelligence and its real-life applications.
• Realms of AI: Computer Vision, Natural Language Processing, and Data Science.
• AI Project Cycle: Problem Scoping, Data Acquisition, Data Exploration, Modelling, Evaluation, Deployment.
• Stakeholder identification and ethical considerations in AI projects.
• Data requirements, features, and basic data visualization.
• Introduction to modeling (rule-based and learning-based).
• Evaluation concepts: true/false positives and negatives.
• Ethics of AI, AI bias, access issues, and impact on society.
• Advantages and disadvantages of AI.
Unit 2: Data Literacy
• Basics of data literacy and its importance.
• Data privacy, security, and associated risks.
• Acquiring, processing, and interpreting data.
• Types of data and methods of data acquisition.
• Data interpretation techniques and importance.
• Fundamentals of data visualization and dashboard creation.
Unit 3: Math for AI (Statistics & Probability)
• Importance of mathematics in AI: pattern recognition and numerical analysis.
• Statistics: definition, real-life applications (e.g., disaster management, sports, healthcare, weather forecasting).
• Data collection and tabulation for statistical analysis.
• Probability: basic concepts, types of events, applications in real-life scenarios (e.g., sports, traffic, weather).
Unit 4: Introduction to Generative AI
• Definition and types of Generative AI.
• Difference between Generative AI and Conventional AI.
• How Generative AI works and where it is applied.
• Benefits, limitations, and ethical considerations of using Generative AI.
Unit 5: Introduction to Python
• Basics of Python programming language.
• Python syntax: variables, operators, expressions, data types (int, float, string), type conversion.
• Input/output functions: print(), input().
• Conditional statements: if, else, elif.
• Loops: for, while.
• Lists: definition and simple operations.
Artificial Intelligence – CLASS X
Unit Topics
Unit 1: Introduction to Artificial Intelligence (AI)
• Understanding human intelligence: reasoning, problem-solving, creativity.
• Defining Artificial Intelligence (AI) and related domains: Data Science, Computer Vision, NLP.
• Applications of AI in real life.
• Introduction to AI ethics: AI bias, data privacy, responsible development.
Unit 2: AI Project Cycle
• Stages of AI Project Cycle: Problem Scoping, Data Acquisition, Data Exploration, Modelling, Evaluation.
• Importance of planning and goal setting in AI projects.
• Types of AI models: Rule-based and Learning-based approaches (Supervised, Unsupervised, Reinforcement Learning).
• Understanding evaluation metrics and importance of model testing.
Unit 3: Advance Python
• Working with Jupyter Notebook and virtual environments.
• Writing basic Python programs using variables, data types, operators, and control structures.
• Using built-in Python functions and libraries for problem-solving.
Unit 4: Data Science
• Defining Data Science and its real-world applications.
• Basics of data acquisition, exploration, and visualization.
• Using Python libraries like NumPy, Pandas, and Matplotlib for analysis.
• Understanding statistical concepts: mean, median, mode, standard deviation.
• Basics of the K-Nearest Neighbour (KNN) algorithm and its application in classification.
Unit 5: Computer Vision
• Introduction to Computer Vision and its applications in AI.
• Concepts of image representation, pixel, resolution, grayscale and RGB images.
• Basics of feature extraction, object detection, and image segmentation.
Unit 6: Natural Language Processing (NLP)
• Understanding Natural Language Processing and its role in AI.
• Applications of NLP: chatbots, sentiment analysis, summarization.
• Language processing challenges: human vs. machine language.
• Concepts of text normalization and Bag-of-Words model in NLP.
