COMPUTER APPLICATION – CLASS IX
Unit Topics
Unit 1: Basics of Information Technology
• Characteristics and components of a computer system: CPU, memory, storage devices, I/O devices.
• Memory types: primary (RAM and ROM), secondary memory.
• Storage devices: hard disk, CD ROM, DVD, flash drive, memory stick.
• I/O devices: keyboard, mouse, monitor, printer, scanner, webcam.
• Types of software: system software (OS, drivers), application software including mobile apps.
• Networking: PAN, LAN, MAN, WAN; wired/wireless, Wi-Fi, Bluetooth, cloud (private/public).
• Multimedia: images, audio, video, animation.
Unit 2: Cyber-safety
• Safe browsing and use of social networks: identity protection, password usage, privacy and confidentiality, cyber stalking, reporting cybercrimes.
• Malware types: viruses, adware.
Unit 3: Office Tools
Word Processor:
• Create and save documents, edit and format text (B, I, U, font type/size/color), alignments.
• Format paragraphs, headers/footers, page numbers, grammar/spell check, sub/superscript, symbols.
• Print preview, insert images, bullets, tables (insert/delete/merge/split), borders and shading.
• Use auto-format, track changes, review comments, drawing tools, shapes, math symbols.

Presentation Tool:
• Understand slide shows, slide layouts, create/save presentations, slide views (normal, sorter, handout).
• Edit slides: titles, subtitles, background, watermark, headers/footers, slide numbers.
• Insert images, animations, sound effects, rehearse timings.

Spreadsheets:
• Worksheet vs. workbook, create/save worksheet.
• Enter data (numbers, text, date/time), use autofill.
• Edit/format sheet (font, size, alignment, colors), insert/delete cells/rows/columns.
• Use formulas (+, −, ×, ÷), refer to cells, print worksheet.
• Apply statistical functions: SUM(), AVERAGE(), MAX(), MIN(), IF().
• Embed charts: line, pie, scatter, bar, area.
COMPUTER APPLICATION – CLASS X
Unit Topics
Unit 1: Networking
• Internet: WWW, web servers, clients, websites, web pages, browsers, blogs, newsgroups, HTML, web addresses, emails, downloading/uploading files.
• Internet protocols: TCP/IP, SMTP, POP3, HTTP, HTTPS.
• Remote login/file transfer: SSH, SFTP, FTP, SCP, TELNET.
• Services: info retrieval, search engines, finding people online.
• Web services: chat, email, video conferencing, e-Learning, e-Banking, e-Shopping, e-Reservation, e-Governance, e-Groups, social networking.
• Mobile technologies: SMS, MMS, 3G, 4G, 5G.
Unit 2: HTML
• Introduction to HTML and webpage design: creating and saving HTML docs, accessing via browser.
• Basic HTML tags: html, head, title, body (text, background, bgcolor, link, vlink, alink), br, hr, comments.
• Formatting tags: h1–h6, p, b, i, u, ul, ol, li, dl, dt, dd, attributes for ol/ul.
• Font tags and attributes: face, size, color.
• Images and media: img (src, width, height, alt), sup, sub.
• Forms: textbox, radio, checkbox, password, list, combobox.
• Embedding audio/video in HTML.
• Tables: table, tr, th, td, rowspan, colspan.
• Links and anchors: href, mailto, targets.
• CSS: color, background-color, border-style, margin, height, width, outline, font (family, style, size), align, float.
Unit 3: Cyber Ethics
• Netiquettes and responsible online behavior.
• Software licenses and open source movement.
• Intellectual property rights, plagiarism, digital property rights.
• Freedom of information and digital divide.
• E-commerce: privacy, fraud, secure data transmission.
