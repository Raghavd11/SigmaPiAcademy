Computer Science – CLASS XI
Unit Topics
Unit 1: Computer Systems and Organisation
• Basic computer organisation: Introduction to computer system, hardware, software, input and output devices, CPU, primary, cache, and secondary memory, memory units (bit to petabyte).
• Types of software: System software (OS, utilities, device drivers), programming tools (assembler, compiler, interpreter), application software.
• Operating System: Functions and user interface of OS.
• Boolean logic: Gates – NOT, AND, OR, NAND, NOR, XOR; truth tables, De Morgan’s laws, logic circuits.
• Number systems: Binary, Octal, Decimal, Hexadecimal and conversions between them.
• Encoding schemes: ASCII, ISCII, Unicode (UTF-8, UTF-32).
Unit 2: Computational Thinking and Programming – I
• Problem-solving steps: Analyzing, algorithm development, coding, testing, debugging; flowcharts and pseudocode; decomposition.
• Basics of Python: Features, “hello world”, script and interactive mode, character set, tokens (keywords, identifiers, literals, operators, punctuators), variables, l-value and r-value, comments.
• Data types: Numbers (int, float, complex), Boolean, sequences (string, list, tuple), None, mapping (dictionary), mutable vs immutable.
• Operators: Arithmetic, relational, logical, assignment, augmented, identity (is, is not), membership (in, not in).
• Expressions, statements, type conversions (explicit & implicit), input/output handling.
• Errors: Syntax, logical, run-time errors.
• Flow of control: Indentation, sequential, conditional, iterative.
• Conditional statements: if, if-else, if-elif-else, flowcharts, programs like absolute value, sorting, divisibility checks.
• Loops: for, range(), while, break, continue, nested loops, programs like pattern printing, summation, factorial.
• Strings: Operations (concatenation, repetition, slicing), traversal, methods like len(), upper(), lower(), strip(), replace(), find(), etc.
• Lists: Indexing, slicing, list methods (append, extend, insert, remove, sort, reverse, etc.), nested lists, frequency and linear search.
• Tuples: Indexing, slicing, methods like count(), index(), sorted(), max(), min(), sum(); tuple assignment and nesting.
• Dictionaries: Key-value access, mutability, traversal, methods like get(), update(), pop(), keys(), values(), etc.
• Python modules: Importing modules (math, random, statistics), using functions like sqrt(), pow(), randint(), mean(), etc.
Unit 3: Society, Law and Ethics
• Digital footprints and responsible digital behavior.
• Digital society and netizen: net etiquette, communication etiquette, social media etiquette.
• Data protection and IPR: Copyright, patent, trademark, plagiarism, open-source licenses (Creative Commons, GPL, Apache).
• Cybercrime: Hacking, eavesdropping, phishing, ransomware, cyberbullying, trolling.
• Cyber safety: Safe browsing, identity protection, confidentiality.
• Malware: Viruses, trojans, adware.
• E-waste management and responsible disposal.
• IT Act overview.
• Social impact of technology: Gender and disability issues in digital access and education.
