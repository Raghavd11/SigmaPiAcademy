COMPUTER APPLICATION – CLASS IX
Unit Topics
Unit 1: Introduction to Object Oriented Programming
1.1:
• Principles of Object Oriented Programming: Data abstraction, Inheritance, Polymorphism, Encapsulation.
• Difference between Procedure Oriented and Object Oriented Programming.

1.2:
• Introduction to JAVA: Types of Java programs – Applets and Applications.
• Java Compilation process: Source code, Byte code, Object code, Java Virtual Machine (JVM).
• Features of Java – Simple, Robust, Secured, Object-Oriented, Platform Independent, etc.
Unit 2: Elementary Concept of Objects and Classes
2.1:
• Modelling entities and their behaviour using objects.
• Class as a blueprint/factory for creating objects.
• Objects encapsulate state (attributes) and behaviour (methods).
• Class as a user-defined data type.
Unit 3: Values and Data Types
3.1:
• Character set, ASCII code, Unicode.
• Escape sequences: \\n, \\t, \\\\, \\", \\\'.
• Tokens: keywords, identifiers, literals, punctuators, operators.
• Constants and Variables, Primitive and Non-Primitive Types.
• Size of primitive types, Implicit and Explicit Type Conversion.
Unit 4: Operators in Java
4.1:
• Forms of Operators: Unary, Binary, Ternary.
• Types: Arithmetic, Relational, Logical, Assignment, Increment, Decrement, Shorthand.
• Counters and Accumulators.
• Operator precedence and associativity, prefix and postfix.
• ‘new’ operator and dot operator.
• Introduction to System.out.println() and System.out.print().
Unit 5: Input in Java
5.1:
• Initialization, Parameters, Packages.
• Scanner Class Methods: nextShort(), nextInt(), nextLong(), nextFloat(), nextDouble(), next(), nextLine(), next().charAt(0).
• Types of Errors: Syntax, Runtime, Logical.
• Comments: Single-line (//) and Multi-line (/* ... */).
Unit 6: Mathematical Library Methods
6.1:
• Math class methods: pow(x, y), sqrt(x), cbrt(x), ceil(x), floor(x), round(x), abs(a), max(a, b), min(a, b), random().
• Use of expressions with Math class and operators.
Unit 7: Conditional Constructs in Java
7.1:
• if, if else, if else if ladder, Nested if.
• switch-case, default, break.
• Fall-through conditions, Menu driven programs.
• System.exit(0) to terminate programs.
Unit 8: Iterative Constructs in Java
8.1:
• Entry controlled loops: for, while.
• Exit controlled loop: do while.
• Loop variations: break, continue, nested, finite/infinite, delays.
• Loop conversions: for ↔ while ↔ do while.
Unit 9: Nested for Loops
9.1:
• Introduction with simple examples.
• break and continue within nested loops.
• Programs: rectangular, triangular (right-angled) patterns, single-variable series.
Unit 10: Computing and Ethics
10.1:
• Ethical issues in computing.
• Intellectual property rights, Privacy, Data protection on the Internet.
• Spam, Software piracy, Cybercrime, Hacking, Malicious code and intent.
• Emphasis on good etiquette and ethical practices.
Unit 11: Internal Assessment
11.1:
• Minimum of 20 programming assignments required.
• Emphasis on design, logic, testing, and application of theory.

Sample assignments:
• Programs with assignment statements (area, volume, swapping, etc.)
• Input via parameters and Scanner class.
• Use of Math methods (sqrt, pow, abs, etc.)
• Conditional programs using if/else/ladder/switch.
• Loops: for/while/do while for patterns, series, prime, palindrome, etc.
• Nested loops and menu-driven programs.
COMPUTER APPLICATION – CLASS X
Unit Topics
Unit 1: Revision of Class IX Syllabus
1.1:
• Introduction to Object Oriented Programming concepts.
• Elementary Concept of Objects and Classes.
• Values and Data Types.
• Operators in Java.
• Input in Java.
• Mathematical Library Methods.
• Conditional Constructs in Java.
• Iterative Constructs in Java.
• Nested for loops.
Unit 2: Class as the Basis of all Computation
2.1:
• Classes and Objects with real-life examples.
• Object encapsulation of state and behaviour.
• Member variables, methods, attributes.
• Primitive and composite data types.
• Object creation and method calls.
Unit 3: User-defined Methods
3.1:
• Method syntax and declaration.
• Static and non-static methods.
• Method overloading, method signature.
• Pure vs impure methods.
• Call by value (with programs).
• Call by reference (definition and example).
• Returning values, modular programming.
Unit 4: Constructors
4.1:
• Definition, characteristics and types of constructors.
• Default and parameterized constructors.
• Constructor overloading.
• Difference between constructor and method.
Unit 5: Library Classes
5.1:
• Wrapper classes, Autoboxing and Unboxing.
• Primitive vs class types with examples.
• Common wrapper methods:
parseInt(), parseFloat(), isDigit(), isLetter(), isUpperCase(), isLowerCase(), toUpperCase(), toLowerCase().
Unit 6: Encapsulation
6.1:
• Access specifiers: private, protected, public.
• Visibility and scope rules.
• Types of variables: class, instance, argument, local.
Unit 7: Arrays
7.1:
• Single and double-dimensional arrays.
• Declaration, initialization, accepting input.
• Accessing elements of arrays.
• Sorting: selection sort, bubble sort.
• Searching: linear search, binary search.
• Matrix operations: sum of row, column, diagonal elements.
Unit 8: String Handling
8.1:
• String class and methods:
trim(), length(), charAt(), indexOf(), lastIndexOf(), substring(), toLowerCase(), toUpperCase(), replace(), startsWith(), endsWith(), equals(), equalsIgnoreCase(), compareTo(), compareToIgnoreCase(), valueOf().
• Programs for extraction, manipulation, palindrome, alphabetical sorting, string search.
