Artificial Intelligence – CLASS XI
Unit Topics
Unit 1: Basic Concepts of AI (i) Definition, evolution, applications, and commonly used AI technologies.
(ii) Role of data and information, binary logic, deterministic vs probabilistic problems with examples.
(iii) Overview of decision making, machine intelligence, and cybersecurity.
(iv) Components of AI project framework: scoping, acquisition, exploration, modeling, evaluation.
(v) Overview of data representation and Python programming basics: data types, variables, operators, control statements, functions.
Unit 2: NLP, ML, and AI in Real Life (i) AI History, machine learning (ML), deep learning (DL), neural networks; applications like image recognition, computer vision, and speech recognition.
(ii) Natural Language Processing (NLP): Text generation, translation, chatbots, search engines, voice assistants.
(iii) Potential use of AI: smartphones, auctions, scanners, e-commerce, social media.
(iv) AI & Society: Applications in healthcare, transportation, disaster prediction, and agriculture.
Unit 3: Mathematics for AI (i) Matrices: Introduction, types, operations (add, subtract, multiply, transpose).
(ii) Vectors: Arithmetic and applications.
(iii) Set Theory: Joins, set operations, relational algebra.
(iv) Statistics: Mean, median, mode, variance, standard deviation.
Unit 4: Data Visualization (i) Using Python: matplotlib, seaborn, Excel; data cleaning with pandas and Excel.
(ii) Graph Types: Bar, histogram, scatter, pie.
(iii) Dimensionality: Representing multi-dimensional data using graphs.
Unit 5: Data Processing (i) Data Cleaning: Using pandas, handling duplicates and inconsistencies.
(ii) Kaggle Datasets: Creating and working with data frames.
(iii) Transformation and Standardization: Prepping data for analysis.
Unit 6: Data Modeling & Regression (i) Types of Data Models: Dimensional, relational, entity-relational.
(ii) Regression: Concepts, dependent vs independent variables, regression types.
(iii) Linear Regression: Least square method, properties, coefficients.
(iv) Solving Equations: Practical applications.
Unit 7: Ethical Practices in AI AI ethics: avoiding bias, ensuring data privacy, mitigating environmental impact.
Importance of ethical AI design and its social consequences.
Artificial Intelligence – CLASS XII
Unit Topics
Unit 1: Applications of AI (i) Natural Language Processing (NLP): Named Entity Recognition (NER), Text Summarization, Spam Email Detection.
(ii) Statistics and Probability: Measures of central tendency and spread, probability basics and common distributions.
(iii) Hypothesis Testing: Null/alternative hypotheses, Type I & II errors, regression analysis, correlation coefficients.
Unit 2: AI Paradigms – NN, ML, DL (i) Artificial Neural Networks (ANN): Neurons, layers, forward/backward propagation, MLP, handwritten digit recognition.
(ii) Machine Learning (ML): Concept, example of house price prediction using decision trees.
(iii) Deep Learning (DL): Neural networks for image/pattern recognition, example using CNN for image classification.
Unit 3: Practical Applications of ANN (i) Data Preparation: NumPy, Pandas, Scikit-learn for preprocessing.
(ii) Building the Network: Using Keras or TensorFlow.
(iii) Training the Network: Fit model using epochs and batch size.
(iv) Model Evaluation: Accuracy, precision, recall; visualization with Matplotlib.
(v) Tuning: Grid/random search for hyperparameter optimization.
Unit 4: Practical Applications of ML (i) Predictive Maintenance: Equipment failure prediction.
(ii) Fraud Detection: Credit card and identity fraud prevention.
(iii) Medical Diagnosis: Analyzing imaging, genetics, and records.
Unit 5: Computer Vision (CV) (i) Introduction to CV: Real-world applications using OpenCV.
(ii) Image Classification: Categorizing objects (e.g., cats vs dogs).
(iii) Object Detection: Identifying and locating objects in images.
(iv) Facial Recognition: Recognizing known faces in images/videos.
(v) Color Detection: Sorting/detecting based on color in various contexts.
(vi) Object Tracking: Monitoring object movement over time in video.
