AI & ROBOTICS – CLASS IX
Unit Topics
Unit 1: Introduction to Robotics (i) Understanding Robots: Definition, characteristics, and benefits.
(ii) Evolution of Robots; Laws of Robotics: Historical context and Asimov's Three Laws.
(iii) Classification of Robots: Field/terrain-based, control-based, and bio-inspired.
(iv) Real-world Robots and Applications: Examples from domestic, industrial, medical, defense, entertainment, and agriculture fields.
Unit 2: Robot as a System (i) Building Blocks: Mechanical, electronic, and computational components.
(ii) Identification: Through videos, graphics, industrial robots, humanoids, mobile robots.
Unit 3: Concepts in Robotics (i) Types of Motion: Linear, angular, circular. Joints: Prismatic, revolute, spherical. Links: Rigid, soft.
(ii) Links and Joints: Motion creation and identification in systems.
(iii) Degree of Freedom: Definition and illustration.
Unit 4: Introduction to Artificial Intelligence (AI) (i) Meaning and Brief History: Definition, 1956 McCarthy, Turing Test.
(ii) Applications and Benefits: Across sectors, everyday examples.
(iii) Ethical Considerations: Bias, fairness, accountability, transparency.
Unit 5: Data, Information & Computing Evolution (i) Types of Data: Audio, visual, numeric, text. Data to Information.
(ii) Evolution of Computing: Binary logic, deterministic vs probabilistic computing.
Unit 6: Programming with Python – Basics (i) Python Introduction: High-level, object-oriented, general-purpose.
(ii) Data Types & Variables: Numeric, Boolean, sequences, sets, dictionaries.
(iii) Operators: Arithmetic, logical, assignment, comparison, identity, membership.
Unit 7: Programming with Python – Control & Functions (i) Conditional Statements: if, if-else, elif, shorthand.
(ii) Control Statements: Loops – for, while, nested, break, continue.
(iii) Functions: Built-in vs user-defined, arguments, return types, modularity.
Unit 8: AI Project Framework (i) AI Concepts: Broad vs narrow AI, expert systems, CV, NLP, NN.
(ii) AI Project Cycle: Problem Scoping, Data Acquisition, Exploration, Modelling, Evaluation.
AI & ROBOTICS – CLASS X
Unit Topics
Unit 1: New Age Robotic Systems Warehouse Robots, Assistant Robots, Smart Homes, Smart Schools, Smart Mobility, Autonomous Cars and Drones, Robotics in Medicine & Healthcare. Use cases like elevators to explain the concept.
Unit 2: From Robots to Cobots (i) Difference between Machines and Robots: Continuous sensing and programmed actions.
(ii) Cobots: Definition, human-in-the-loop systems, multi-robot interaction, significance and evolution from robots.
Unit 3: Components of Robots as a System (i) Gears: Principles, transmission, amplification of force, gear types and ratios.
(ii) Sensors: Types (vision, tactile, motion, speech), classification (internal/external), and functions.
(iii) Actuators: Linear and rotary, with examples like joints and wheels.
(iv) Controllers: Control systems in robotics, manual vs automatic, block diagrams (input-controller-robot-feedback).
(v) Integration: Use case of angular position via sensors and motors in robotic arms.
Unit 4: Visualization, Design and Creation of Components (i) Mechanical Block: Visualization and design using Tinkercad; revolute and prismatic joints, RR mechanisms.
(ii) Visualization of Motion: Tinkercad-based simulations of designed motion.
Unit 5: Integrating Robots as a System Building simple robotic systems: Wheeled mobile robots, RR Mechanism, use of SBC (Single Board Computers), Tinkercad integration.
Unit 6: Decision Making in Machines (i) Automated vs Autonomous Systems: Deterministic vs probabilistic nature.
(ii) Human vs Machine Decision-Making: Subjective vs objective classification.
(iii) Machine Learning: Introduction, steps, importance of data and programming, e.g., fruit sorting.
Unit 7: Machine Intelligence and Cybersecurity (i) Machine Intelligence: Comparison with human intelligence, Turing Test.
(ii) Cybersecurity: Ethical concerns—hardware theft, software piracy, fraud, hacking, viruses, accountability.
Unit 8: AI Project Framework (i) Problem Scoping: 4 Ws (Who, What, Where, Why), problem statement.
(ii) Data Acquisition: Features, sources, training/testing data, system maps, Kaggle datasets.
(iii) Data Exploration: Data visualization – bar graphs, line charts, scatter plots, pie charts.
(iv) Modelling and Evaluation: Linear regression, rule-based, supervised/unsupervised learning.
Unit 9: Programming with Python – Advanced (i) Modules & Packages: NumPy, Pandas, Matplotlib; syntax and use cases.
(ii) Lists & Tuples: Creation, manipulation, access, conversion.
(iii) Strings: Operations, functions (len, upper, lower, replace, find, join, etc.), indexing and slicing.
