Computer Science – CLASS XI
Unit Topics
Unit 1: Numbers and Number Systems Representation and interconversion of numbers in binary, octal, decimal, and hexadecimal systems.
Addition and subtraction using 1’s and 2’s complement, positional systems, and carry operations.
Unit 2: Encodings and Representations (a) Binary encoding for integers and real numbers: sign-magnitude, 2’s complement, mantissa-exponent notation.
(b) Character encodings: ASCII, ISCII, Unicode and their application in local languages.
Unit 3: Logic and Hardware (a) Propositional logic, well-formed formulae, truth values, and truth tables.
(b) Logic gates: AND, OR, NOT, NAND, NOR, XOR, XNOR; half adder, full adder; hardware implementation.
Unit 4: Object-Oriented Programming (OOP) with Java JVM, compile-time and run-time errors, exception handling, classes and objects, methods and constructors.
BlueJ IDE, analysis of real-world examples, and Java execution model.
Unit 5: Java Basics and Data Types Primitive types and wrapper classes, type casting, variables, expressions, and operators.
Concepts of memory locations and references in variable assignment.
Unit 6: Control Structures Conditional statements (if, if-else, switch-case), loops (for, while, do-while), continue, break, scope and visibility rules.
Unit 7: Methods and Constructors Method definitions, side effects, static methods and variables, use of this, overloading constructors.
Unit 8: Arrays and Strings One and two-dimensional arrays, common algorithms (search, sort, max/min), Java String class methods.
String manipulation techniques (substring, concatenation, indexOf, replace).
Unit 9: Input/Output and File Handling Scanner and Printer classes, tokens, delimiters, StringTokenizer, text and binary file operations using streams.
Unit 10: Recursion Recursive functions, base cases, examples (factorial, GCD, binary search, base conversion), comparison to iterative logic.
Unit 11: Problem Solving and Implementation Lab assignments, algorithm design, debugging, correctness validation, implementation of Java algorithms.
Unit 12: Packages in Java Creation and importing of user-defined packages, Java API, object interaction across packages.
Unit 13: Emerging Trends and Ethics Overview of AI, IoT, VR/AR, cybersecurity, privacy, netiquette, IP rights, software licensing, Free Software Foundation.
Discussion on societal and ethical implications of computing.
Computer Science – CLASS XII
Unit Topics
Unit 1: Applications of AI (i) Natural Language Processing (NLP): Named Entity Recognition (NER), Text Summarization, Spam Email Detection.
(ii) Statistics and Probability: Measures of central tendency and spread, probability basics and common distributions.
(iii) Hypothesis Testing: Null/alternative hypotheses, Type I & II errors, regression analysis, correlation coefficients.
Unit 2: AI Paradigms – NN, ML, DL (i) Artificial Neural Networks (ANN): Neurons, layers, forward/backward propagation, MLP, handwritten digit recognition.
(ii) Machine Learning (ML): Concept, example of house price prediction using decision trees.
(iii) Deep Learning (DL): Neural networks for image/pattern recognition, example using CNN for image classification.
Unit 3: Practical Applications of ANN (i) Data Preparation: NumPy, Pandas, Scikit-learn for preprocessing.
(ii) Building the Network: Using Keras or TensorFlow.
(iii) Training the Network: Fit model using epochs and batch size.
(iv) Model Evaluation: Accuracy, precision, recall; visualization with Matplotlib.
(v) Tuning: Grid/random search for hyperparameter optimization.
Unit 4: Practical Applications of ML (i) Predictive Maintenance: Equipment failure prediction.
(ii) Fraud Detection: Credit card and identity fraud prevention.
(iii) Medical Diagnosis: Analyzing imaging, genetics, and records.
Unit 5: Computer Vision (CV) (i) Introduction to CV: Real-world applications using OpenCV.
(ii) Image Classification: Categorizing objects (e.g., cats vs dogs).
(iii) Object Detection: Identifying and locating objects in images.
(iv) Facial Recognition: Recognizing known faces in images/videos.
(v) Color Detection: Sorting/detecting based on color in various contexts.
(vi) Object Tracking: Monitoring object movement over time in video.
Unit 6: Boolean Algebra (a) Propositional logic, wffs, truth tables, valid formulae, equivalence laws (De Morgan, implication, etc.), logical connectives, tautology, contradiction, converse, inverse, contraposition.
(b) Binary values, postulates of Boolean algebra, AND, OR, NOT; truth tables.
(c) Theorems of Boolean algebra, Karnaugh maps, maxterms/minterms, canonical forms, simplification, adders, majority circuits.
Unit 7: Computer Hardware (a) Logic gates (NOT, AND, OR, NAND, NOR, XOR, XNOR), applications in circuits.
(b) Boolean applications: half/full adders, encoders, decoders, multiplexers. NAND/NOR as universal gates.
Unit 8: Algorithms and Java Implementation Algorithmic problem solving, Java syntax and debugging, assignments aligned with major topics, structured programming using BlueJ or similar IDEs.
Unit 9: Java Programming Concepts Objects and classes, constructors, input/output (Scanner, Printer), tokens, exceptions, wrapper classes, type casting, variables, expressions, and assignment.
Conditional and looping statements, scope, methods (arguments, static, this), abstraction, algorithmic methods.
Unit 10: Structured Data Types and Strings Arrays (1D & 2D), sorting (bubble, selection, insertion), String class, StringBuffer, operations like substring, indexOf, replace, concatenation, structured algorithms.
Unit 11: Recursion Recursive functions (factorial, GCD, binary search), number base conversions, Tower of Hanoi, concept of base case.
Unit 12: OOP Concepts – Inheritance & Interfaces Inheritance, superclass-subclass relationships, abstract classes, dynamic binding, interface implementation and applications, subclass polymorphism.
Unit 13: Data Structures Stacks, queues, deques, linked lists (insert/delete/reverse/extract), infix-postfix-prefix, binary trees (concepts and traversals: in, pre, post-order).
Unit 14: Computational Complexity Input size, best/worst/average case, time estimation, Big O notation, analyzing searching/sorting/linear systems algorithms.
